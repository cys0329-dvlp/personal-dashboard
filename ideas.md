# 개인 대시보드 디자인 아이디어

## 아이디어 1: 미니멀 페이퍼 플래너
<response>
<text>
**Design Movement**: 스칸디나비안 미니멀리즘 + 종이 질감 플래너
**Core Principles**:
- 오프화이트 배경에 연필 선 같은 얇은 경계선
- 손글씨 느낌의 폰트와 깔끔한 산세리프 조합
- 여백을 최대한 활용한 호흡감 있는 레이아웃
- 색상은 최소화하고 강조점에만 사용

**Color Philosophy**: 크림색(#FAF8F3) 배경, 차콜(#2C2C2C) 텍스트, 포인트 색상으로 테라코타(#C4704F)와 세이지 그린(#7A9E7E) 사용. 종이 위에 쓴 듯한 아날로그 감성.

**Layout Paradigm**: 좌측 고정 사이드바(아이콘+텍스트), 우측 메인 콘텐츠 영역. 콘텐츠는 카드 없이 구분선으로만 섹션 구분.

**Signature Elements**:
- 점선 구분선과 체크박스 스타일 할 일 목록
- 캘린더에 날짜별 점(dot) 인디케이터

**Interaction Philosophy**: 호버 시 배경색이 살짝 변하는 부드러운 전환, 클릭 시 미세한 눌림 효과

**Animation**: 페이지 전환 시 슬라이드-인, 모달 오픈 시 페이드+스케일

**Typography System**: Playfair Display (제목) + Noto Sans KR (본문)

## 아이디어 2: 모던 다크 대시보드
<response>
<text>
**Design Movement**: 사이버펑크 미니멀 + 데이터 시각화 중심
**Core Principles**:
- 딥 네이비 배경에 네온 액센트
- 데이터 중심 레이아웃, 차트와 숫자 강조
- 유리형태(glassmorphism) 카드 컴포넌트
- 그리드 기반 정보 밀도 높은 레이아웃

**Color Philosophy**: 딥 네이비(#0D1117) 배경, 사이언(#00D4FF)과 퍼플(#7C3AED) 액센트, 화이트 텍스트. 데이터가 빛나는 느낌.

**Layout Paradigm**: 상단 헤더 네비게이션 + 3열 그리드 대시보드. 위젯 형태로 각 기능 배치.

**Signature Elements**:
- 글로우 효과가 있는 숫자/금액 표시
- 반투명 유리 카드에 블러 배경

**Interaction Philosophy**: 호버 시 카드가 살짝 떠오르는 효과, 데이터 업데이트 시 숫자 카운트업 애니메이션

**Animation**: 카드 입장 시 스태거 애니메이션, 차트 데이터 드로잉 효과

**Typography System**: Space Grotesk (제목) + Inter (본문)
</text>

## 아이디어 3: 웜 어스톤 생산성 대시보드
<response>
<text>
**Design Movement**: 바우하우스 기능주의 + 따뜻한 어스톤 팔레트
**Core Principles**:
- 기능성과 심미성의 균형
- 따뜻한 베이지/크림 계열로 눈의 피로 최소화
- 좌측 사이드바 + 메인 콘텐츠의 비대칭 레이아웃
- 정보 계층 구조를 색상 농도로 표현

**Color Philosophy**: 따뜻한 크림(#FDF6EC) 배경, 딥 브라운(#3D2B1F) 텍스트, 앰버(#D97706) 포인트, 소프트 그린(#65A30D) 성공 상태. 자연스럽고 편안한 작업 환경.

**Layout Paradigm**: 좌측 240px 사이드바(아이콘+레이블) + 우측 메인 영역. 메인 영역은 탭 기반으로 캘린더/가계부/프로젝트 전환.

**Signature Elements**:
- 둥근 모서리 카드에 따뜻한 그림자(amber 계열)
- 진행 상태 바와 카테고리 색상 도트

**Interaction Philosophy**: 부드러운 호버 전환(200ms), 클릭 피드백으로 미세한 스케일 다운

**Animation**: 탭 전환 시 슬라이드 페이드, 모달 등장 시 바텀-업 슬라이드

**Typography System**: Pretendard (전체) - 한국어 최적화, 굵기 변화로 계층 구분
</text>

## 선택된 디자인: 아이디어 3 (웜 어스톤 생산성 대시보드)
- 따뜻한 어스톤 팔레트로 장시간 사용해도 눈이 편안함
- 한국어에 최적화된 Pretendard 폰트
- 좌측 사이드바로 세 기능(캘린더, 가계부, 프로젝트) 간 빠른 전환
- 기능성과 심미성의 균형 잡힌 바우하우스 스타일
